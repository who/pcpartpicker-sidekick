# Ortus

This project was bootstrapped with [Ortus](https://github.com/who/ortus).

## What is Ortus?

Ortus is a project template that sets up:
- **Beads** (`bd`) - Local-first issue tracking
- **Ralph** - Autonomous task execution loop
- **Interview** - PRD generation through conversation

## Scripts

| Script | Purpose |
|--------|---------|
| `./ortus/ralph.sh` | Run the task implementation loop |
| `./ortus/idea.sh` | Quick-start a new feature |
| `./ortus/interview.sh` | Generate PRD through interview |
| `./ortus/tail.sh` | Watch ralph logs |

## Prompts

The `prompts/` folder contains instructions for Claude:
- `ralph-prompt.md` - Task execution instructions
- `interview-prompt.md` - PRD interview flow
- `prd-prompt.md` - PRD generation template
